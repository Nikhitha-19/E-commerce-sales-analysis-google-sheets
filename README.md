# 🛒 E-Commerce Data Analysis using Google Sheets

## 📌 Project Overview

This project analyzes an E-Commerce Sales dataset using **Google Sheets**. The objective is to transform raw sales data into meaningful business insights by applying spreadsheet formulas, Pivot Tables, and charts.

The project demonstrates practical spreadsheet skills including data preparation, aggregation, summarization, and visualization, making it suitable for beginners and aspiring Data Analysts.

---

## 🎯 Objectives

- Prepare and clean sales data
- Calculate revenue-based metrics using formulas
- Generate business summaries using aggregation functions
- Build Pivot Tables for data analysis
- Create charts to visualize key business insights

---

## 🛠️ Tools Used

- Google Sheets
- Spreadsheet Formulas
- Pivot Tables
- Charts (Bar Chart & Pie Chart)

---

## 📂 Dataset Information

The dataset contains order-level information including:

- Order ID
- Product Name
- Category
- Region
- Sales Representative
- Quantity
- Unit Price
- Status (Shipped, Pending, Cancelled)

---

# 📋 Project Tasks

## Task 1 – Data Preparation

Created new calculated columns using formulas:

### Total Revenue

Calculated by:

```
=Quantity * Unit Price
```

### Order Type

Classified each order using an IF formula:

- High Value → Total Revenue > 500
- Standard Value → Total Revenue ≤ 500

---

## Task 2 – Business Metrics

Created a summary table containing:

| Metric | Value |
|---------|---------:|
| Total Revenue | $1,451,015 |
| Completed Revenue (Shipped) | $887,097 |
| Pending Orders | 195 |

---

## Task 3 – Pivot Table Analysis

### 1. Regional Profitability

Summarized Total Revenue by Region and sorted from highest to lowest.

| Region | Revenue |
|---------|----------:|
| Central | $310,901 |
| West | $304,410 |
| East | $299,997 |
| North | $280,164 |
| South | $255,543 |

---

### 2. Product Performance by Status

Analyzed the count of product orders grouped by order status.

This helps identify product demand and order distribution across:

- Shipped
- Pending
- Cancelled

---

### 3. Representative Sales by Order Type

Calculated Total Revenue generated by each Sales Representative based on:

- High Value Orders
- Standard Value Orders

This analysis helps compare sales performance among representatives.

---

## Task 4 – Data Visualization

Created two charts to represent business insights:

### 📊 Bar Chart

**Total Revenue by Category**

Categories:

- Electronics
- Accessories
- Gaming
- Peripherals

**Insight**

Electronics generated the highest revenue, significantly outperforming all other categories.

---

### 🥧 Pie Chart

**Order Status Distribution**

- Shipped — 60.5%
- Cancelled — 20.0%
- Pending — 19.5%

**Insight**

Most orders were successfully shipped, while pending and cancelled orders accounted for less than 40% of total orders.

---

# 📈 Key Insights

- Total revenue generated was **$1.45 Million**.
- Electronics contributed the majority of sales revenue.
- Central region recorded the highest revenue.
- More than **60%** of orders were successfully shipped.
- High-value orders accounted for the majority of total revenue.
- Sales representatives showed varying performance based on high-value sales.
